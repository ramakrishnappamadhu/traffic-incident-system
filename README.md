
# Traffic Incident Analysis System

Built on Red Hat OpenShift AI after the Feb 2026 workshop.

## Pipeline

1. YOLOv8 detects vehicles in traffic images
2. RAG retrieves California traffic laws
3. System generates incident reports with legal citations

## Technologies

- YOLOv8, LangChain, PyMilvus, Sentence Transformers
- Red Hat OpenShift AI Developer Sandbox
